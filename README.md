# Paranã Turismo

Portal de turismo de Paranã (TO): catálogo premium, classificados locais, atrativos e painel do lojista.

## Stack

- Next.js 16 + TypeScript + Tailwind CSS
- Deploy: Vercel
- Próximo: Supabase (Auth/DB) + Mercado Pago (VIP + Pix bump)

## Desenvolvimento

```bash
npm install
npm run dev
```

Abra [http://localhost:3000](http://localhost:3000).

## Scripts

- `npm run dev` — servidor local
- `npm run build` — build de produção
- `npm run start` — servir build
